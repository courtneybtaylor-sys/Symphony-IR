# Getting Started with Symphony-IR

Choose your platform:

## Windows
Run the Windows installer from the Releases page. The installer will guide you through setup.

## macOS
1. Download the DMG file from Releases
2. Drag Symphony-IR to Applications
3. Run Symphony-IR from Applications

## Linux
1. Download the AppImage from Releases
2. Make it executable: `chmod +x Symphony-IR-*.AppImage`
3. Run it: `./Symphony-IR-*.AppImage`

## Configuration
On first run, the setup wizard will guide you through:
- Setting your API key
- Selecting your AI model
- Configuring advanced options

Or run: `python -m ai-orchestrator --setup`
